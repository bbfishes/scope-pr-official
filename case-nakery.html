品牌溝通

家樂福 Nakery 裸焙坊 —— 從門市開幕到美食名人合作的全方位品牌行銷

標籤：#品牌溝通 #記者會 #校園行銷 #名人合作 #新品上市 #KOL行銷

【背景：為家樂福自有烘焙品牌打造市場定位】

家樂福推出自有烘焙品牌 Nakery 裸焙坊，以「低糖、無添加」為核心訴求，瞄準注重健康的消費者市場。Scope PR 受邀擔任品牌上市與後續系列活動的公關顧問，挑戰在於：如何讓自有烘焙品牌在競爭激烈的烘焙市場中建立清晰定位，並透過系列活動累積品牌資產。

【策略：分階段品牌建設，整合線上線下資源】

Scope PR 以「階段性品牌建設」為核心策略，為 Nakery 打造從上市到深化價值的完整行銷路徑：

  • 品牌上市記者會：2022 年西屯店開幕時，統籌「開幕暨低糖麵包分享會」記者會，邀請營養師李婉萍專業職人站台，透過新聞稿與媒體邀約建立首波聲量。
  • 校園巡迴深化認知：2023 年執行「Nakery 無添加 2.0 校園巡迴行銷」，深入校園傳遞健康烘焙理念，培養年輕世代的品牌認知。
  • 名人美食家合作破圈：2024 年推出「作家買菜」系列，攜手馬世芳、詹宏志、楊馥如、王嘉平等四位知名美食作家，於家樂福門市舉辦職人分享會，將品牌理念與名人影響力結合，吸引高端消費族群。
  • 節令新品持續發聲：配合節令推出青森紅玉蘋果派、鳳梨酥等新品上市記者會，維持品牌話題熱度並擴展產品線聲量。

【執行亮點：專業職人與跨界名人雙軌並進】

本系列專案橫跨 2022 至 2024 年，Scope PR 以「專業職人 + 跨界名人」雙軌策略，為 Nakery 建立起「健康、專業、有溫度」的品牌形象：

  • 攜手日本甜點大師小山進，借勢國際職人背書強化產品專業形象。
  • 邀請營養師李婉萍等專業職人，以科學數據支持「低糖、無添加」產品定位。
  • 透過內容企劃與敘事策略，將產品置入生活場景，提升消費者認同感。
  • 搭配校園巡迴、KOL 分享會等多元形式，覆蓋不同年齡層與消費族群。

【成果：多年合作，建立品牌堅實基礎】

  • • 品牌聲量持續累積：透過多場記者會與名人活動，協助 Nakery 建立穩定的媒體曝光與社群討論度。
  • • 多元族群覆蓋：從校園年輕族群到高端美食愛好者，成功拓展不同消費市場的品牌認知。
  • • 品項延伸能力：從低糖麵包到蘋果派、鳳梨酥等節令新品，展現 Scope PR 協助品牌持續擴展產品線的行銷執行力。
  • • 長期夥伴關係：多年服務期間，持續獲得家樂福信任，驗證 Scope PR 在食品零售行銷領域的專業深度。
